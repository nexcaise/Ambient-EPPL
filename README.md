# Ambient - EPPL(Extends Piston Push Limits)

> **Signatures/Patterns discovered by [Max-RM](https://github.com/Max-RM).**

> **Original Repository: [Extend-PistonPushLimit-for-MCBE](https://github.com/Max-RM/Extend-PistonPushLimit-for-MCBE).**

With this mod you can push blocks using pistons or sticky pistons with the ability to push blocks more than 13+

## Requirements

* Ambient — <https://ambient.kitsuri.dev>
* libEPPL mod installed

## Installation

1. Download libEPPL from [Releases](https://github.com/NexCaise/Ambient-EPPL/releases)
2. Install Ambient
3. Add libEPPL mod in Ambient
4. Launch Minecraft through Ambient
